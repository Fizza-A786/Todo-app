✨ Cute Todo App (React + Vite)

A beautiful, responsive and aesthetic Todo App built using React + Vite and Tailwind CSS.
Designed with a soft pink & purple girly UI, smooth animations, and mobile-first experience 📱💖

🌸 Features
➕ Add new tasks instantly
✔ Mark tasks as completed
❌ Delete tasks easily
💖 “Completed” badge with strike-through effect
🌗 Dark / Light mode toggle
📱 Fully responsive (mobile + desktop)
✨ Smooth animations & modern UI
🎀 Cute pastel aesthetic design
